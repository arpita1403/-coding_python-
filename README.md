Binary Tree Traversal in Python

This program implements a binary tree in Python and performs
Inorder, Preorder, and Postorder traversals.

Tree Structure:

        1
       / \
      2   3
     / \
    4   5

Traversals:

Inorder Traversal:
Left -> Root -> Right

Preorder Traversal:
Root -> Left -> Right

Postorder Traversal:
Left -> Right -> Root

How to Run:

1. Make sure Python is installed.
2. Run the file using:
   python binary_tree.py

Output:

Inorder Traversal:
4 2 5 1 3

Preorder Traversal:
1 2 4 5 3

Postorder Traversal:
4 5 2 3 1
